# Node.js GCODE sender

## Introduction
This project is design to work w/ a GRBL Plotter and a Raspberry Pi 4 during the Conciergerie exhibition "Algorithmes" on 09-24-2020.

## Requirements
- Raspberry Pi 4 on Debian
- GRBL Plotter (GKDraw Pro X3)
- GPIO button
- node.js
